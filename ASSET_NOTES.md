# Asset notes

## Generated visual assets

The following original visual-development assets were generated for Digi World with OpenAI image generation and then integrated as runtime textures:

- `public/assets/digital-cathedral.png` — a cinematic, photoreal digital-cathedral panorama with obsidian architecture, cyan energy seams, atmospheric depth, and an original high-tech visual language.
- `public/assets/obsidian-armor-panels.png` — a seamless dark obsidian/carbon armor-panel material with realistic micro-scratches, machined seams, restrained cyan circuitry, and no text or logos.

Both were generated in the built-in image-generation mode and are kept in their original generated form in the Codex generated-images directory as well as in the project's `public/assets` folder.

## Third-party reference asset

`public/assets/models/sentinel-soldier.glb` is based on the `Soldier.glb` example distributed in the official [three.js repository](https://github.com/mrdoob/three.js/tree/dev/examples/models/gltf). Digi World applies its own material treatment, scale normalization, circuitry, disc prop, and gameplay animation blending at runtime.

The three.js source repository is MIT licensed. Media files in example repositories may have separate provenance; verify the upstream model's media rights before commercial distribution or replace it with a commissioned production character.
